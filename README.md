# Hoddies District

Hoddies District is a responsive hoodie showcase website with product sections, pricing in rupees, and local product images.

## Project Structure

- `styles.css` - website styling
- `script.js` - product and interaction logic
- `assets/hoodies/` - local hoodie images

## Run Locally
